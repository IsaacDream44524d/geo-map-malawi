About This Project

This project is part of MUST GIS SOCIETY mappng competition, created as a competition entry to view, analyze and assess spatial data in a simple and interactive way as such visually presenting spatial data using modern technologies.

The project focuses on key themes CLIMATE, ENERGY, RESOURCES AND EARTH SCIENCE giving users a new way to explore patterns and relationships across Malawi.

Built using Python and web technologies, this is not just a technical exercise, a competition trial but an attempt for me to practice my programming and problem solving in the real wolrd.

Developed as part of MUST-GIS SOCIETY competition project.
